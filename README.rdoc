== README

This README would normally document whatever steps are necessary to get the
application up and running.

TO INSTALL

1. git clone https://github.com/ReviAprialdi/robotic-test-software-seni.git
2. cd robotic
3. run bundle install

TO RUN

1. cd robotic
2. run "ruby robotic/main.rb"

IF RUNNING

INPUT :

PLACE 1,2,EAST 
MOVE
MOVE
LEFT
MOVE
REPORT

OUTPUT
3,3,NORTH

Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
